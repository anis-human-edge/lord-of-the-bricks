Reference · How scoring really works

The four engines and what they actually score.

Read this once. Refer back forever. Every artefact we build, every line we rehearse, every hand we shake should map back to one of these four engines.

The four ranks.

Verbatim from the official rubric structure
01Robot Game Rank

Pure mission points.

Up to 3 matches, highest match counts. Robot performance only. No judging. No theatre, no rubric, no charm. Either the robot does the missions or it doesn't.

→ Brick 01 · Performance — see mission plan
02Innovation Project Rank

10 scores (1–4) on the IP rubric.

Five sections: Identify, Design, Create, Iterate, Communicate. Each section has two criteria. Sum gives the IP rank. Five of those ten criteria also feed Core Values — they are dual-marked.

→ Brick 02 · Innovation — see presentation hub
03Robot Design Rank

10 scores (1–4) on the RD rubric.

Same five-section structure as IP. Sum gives the RD rank. Another five of those ten criteria also feed Core Values — also dual-marked.

→ Brick 03 · Robot Design — see presentation hub
04Core Values Rank

The weave.

Five dual-marked criteria from IP + five dual-marked criteria from RD + three Gracious Professionalism scores. Core Values is not a separate session. It is observed and scored through:

  • The IP and RD presentations
  • Volunteer interactions throughout the event
  • Pit behaviour
  • Table behaviour
  • Team conduct across the whole weekend
→ Brick 04 · Core Values — see weave map

Two non-obvious leverage points.

Leverage 01 · The Exceeds column

Score 4 requires a written judge comment.

So every row needs a quotable line or visible artefact from us that makes it easy for the judge to write that comment. No artefact = no Exceeds.

Action: every rubric criterion gets a paired artefact in our kit — a printed card, a one-liner in the script, a visible thing on the table. See judge psychology & Exceeds-trigger artefacts.

Leverage 02 · The Final Share & Feedback slot

Up to 8 minutes — where most teams collapse.

It's also where Core Values and Gracious Professionalism get cemented in the judges' minds. We will weaponise this slot.

Action: a fully-scripted, fully-rehearsed Final Share. See Final Share script.
