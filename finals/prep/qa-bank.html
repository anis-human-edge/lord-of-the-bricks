Robot Design · 10 criteria.

Same shape — fill the answer column

RD · Identify · strategyWhy these missions and not others?

[ Highest-yield logic. Risk profile. Skipped missions and why. ]

Owner: —Draft

RD · Design · build planningWhy this attachment design?

[ The trade-off. What we considered. Why this won. ]

Owner: —Draft

RD · Design · code planningHow does the code decide what to do?

[ Sensor / condition / branch. Show the printed snippet. ]

Owner: —Draft

RD · Create · build qualityShow me how this part works.

[ Hands-on demo. Concrete walk-through. ]

Owner: —Draft

RD · Create · code qualityWho wrote this part of the code?

[ Named kid speaks. They explain it in their own words. ]

Owner: —Draft

RD · Iterate · testingHow do you know it works?

[ Test protocol. Run count. Success rate. Confidence interval if any. ]

Owner: —Draft

RD · Iterate · improvementWhat's a version that didn't work?

[ Specific failed prototype. The reason. The fix. ]

Owner: —Draft

RD · Communicate · explanationCan you explain that without the slides?

[ Practice this — judges will sometimes ask the kids to talk to the robot, not the screen. ]

Owner: —Draft

RD · Communicate · teamworkHow did you handle disagreements?

[ A real example. Names. Process. Outcome. ]

Owner: —Draft

RD · Communicate · clarityWhat would you change for next year?

[ One specific thing. Don't say "everything". ]

Core Values prompts.

Often dropped into IP / RD Q&A

CV · inclusionWho came up with the idea?

[ Honest answer. If it was one kid, name them and explain how the team built around it. Don't pretend it was unanimous. ]

Owner: —Draft

CV · inclusionDid everyone agree on the design?

[ No. Name the disagreement. Name the resolution. ]

Owner: —Draft

CV · respectWho helped you who isn't on the team?

[ Name parents, mentors, partners, schools, community. By name. ]

Owner: —Draft

CV · curiosityWhat surprised you most this season?

[ Personal answer from one kid. Specific. ]

Owner: —Draft

CV · impactIf you had unlimited time, what would you do next?

[ Honest, specific, scoped. Not "save the world" — name the next concrete thing. ]

Owner: —Draft

Meta & curveballs.

The questions that come from outside the rubric

META · graceHow are you finding the competition so far?

[ Specific positive observation. Avoid generic "amazing". Name a team or volunteer. ]

Owner: any kidDraft

META · honestyWhat's hard about this?

[ Honest. One specific hard thing. Don't pretend nothing is. ]

Owner: —Draft

META · curveballIf you could ask us a question, what would it be?

[ A genuine question. Pre-prepared but not robotic. This question, asked back to us, is gold — we should always be ready for it. ]
