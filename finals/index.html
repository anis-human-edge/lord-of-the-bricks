The four bricks.

Each brick weaves into the others.
Brick 01 · Performance

Robot Game

Pure mission points from up to three matches. Highest match counts. Robot performance only — no judging, no theatre. Mission plan, run order, contingency strategy.

Innovation Project

5-minute presentation. Sum of 10 scores (1–4) on the IP rubric: Identify, Design, Create, Iterate, Communicate. Five of those criteria also feed Core Values.
